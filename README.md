# kubejs

Tech modpack featuring Create and Immersive Engineering. Journey across the solar system and beyond as you advance through the ages of human technology, from flint tools and crude furnaces to integrated circuits and interstellar rockets.

## Features
- Curated progression through 5+ technology tiers
- Comprehensive quest book with unique rewards
- Custom ore processing and chemistry based on [Immersive Geology](github.com/Immersive-Geology-Team/Immersive-Geology/)
- Non-linear progression options - multiple ways to produce or process one resource
- Hundreds of custom recipes and many custom materials

This pack is still very much in development and not fit for any kind of release.
